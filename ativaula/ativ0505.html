Questão 05

  • Análise
    1. O código não possui tag Header para cabeçalho.
    2. O código não possui uma tag Main para delimitar o conteúdo principal, onde se encontra a id principal.
    3. Não possui span, p, h2 (poderia ser utilizado).
    4. Uso de br, que é uma má prática visto que o CSS pode fazer o mesmo efeito e de maneira mais flexível.
    5. O uso da tag b, a qual não é mais utilizada, por conta da existência de outras técnicas como strong ou font weight no CSS.
    6. O texto da id principal não está em nenhuma tag.
    7. o estilo CSS da id cabeçalho está inline, quando deveria ser um arquivo separado.
    8. A tag da imagem está como inline também, quando deveria ser um arquivo separado.
    9. a tag i, similar a b pode ser feita em css, dando mais flexibilidade ao código.
